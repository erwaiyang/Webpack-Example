# How to Use

## 1. git clone
<pre>
	$ git clone https://github.com/erwaiyang/Webpack-Example.git
</pre>

## 2. install webpack and webpack dev server
<pre>
	$ sudo npm install -g webpack webpack-dev-server
</pre>

## 3. npm install
<pre>
	$ npm install
</pre>

## 4. run
<pre>
	$ npm run dev
</pre>

## 5. open your browser
`http://localhost:8080/` or the local url it tells you.
